## System Description

Foody is a web-based Online Food Ordering System developed to provide a simple and convenient platform for customers to browse food items, place orders, make payments, and track orders in real time.
The system also provides an administrative dashboard for managing users, food items, and customer orders. Firebase Authentication and Cloud Firestore are used to provide secure user authentication and real-time database functionality.

## Installation / Setup

1. Clone the GitHub repository.

```bash
git clone https://github.com/Althaf-devcode/Foody.git
```

2. Open the project using Visual Studio Code.

3. Configure Firebase by adding your Firebase project configuration to **firebase.js**.

4. Install the **Live Server** extension in Visual Studio Code.

5. Open the project folder.

---

## How to Run

1. Open the project in Visual Studio Code.
2. Install the Live Server extension.

3. Right-click **index.html**.

4. Select **Open with Live Server**.

5. The application will open in your default web browser.
